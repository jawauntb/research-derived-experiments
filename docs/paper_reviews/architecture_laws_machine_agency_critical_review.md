# Critical Review: Architecture Laws for Machine Agency

Date: 2026-07-06

## Verdict

Useful synthesis artifact. The note does not add new empirical evidence, but it
does what the larger metric-stack paper is too large to do quickly: names the
portable architecture laws and states the next major contribution opportunity.

## Main Issues

1. **Keep it bounded.** The note must stay clear that these are minimal
   computational precursors, not consciousness or general-agency proofs.
2. **Use virtual governor only as framing.** Lyons, Pio-Lopez, and Levin (2026)
   is a non-peer-reviewed preprint; cite it only for the signal-architecture
   vocabulary.
3. **Make the laws operational.** Each law should imply a testable gate or
   ablation, not just a slogan.
4. **Bridge to long horizon.** The strongest next opportunity is testing whether
   these laws survive when memory must reach future tool/action commitments.
5. **Absorb structure-compatible results without overclaiming.** The new
   transformation work belongs here as a finite-domain law about OOD pressure,
   not as a universal alignment result.
6. **Avoid duplicating the metric-stack paper.** This artifact should be short,
   strategic, and readable as a roadmap.

## Rewrite Applied

- Added a critical caveats section and a major contribution section focused on

- Added the latest dispatch-surface and structure-compatible transformation

## Contribution Opportunity

The major contribution is a reusable research method:

> For every proposed agent capability, find the proxy that lets behavior pass
> without the intended structure, then add the smallest architecture change that
> makes the structure load-bearing.

That method can guide the next L4 suites across long-horizon tool agents,
structure-compatible generalization, and learned intervention discovery.
