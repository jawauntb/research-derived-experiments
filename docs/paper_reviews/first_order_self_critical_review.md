# Critical Review: First-Order Self Paper

Date: 2026-07-06

## Verdict

Revise and sharpen. The paper is already one of the program's strongest negative
results because it separates action competence from self/world attribution. Its
weakness is that the architecture lesson is present but not yet named as a
portable law.

## Main Issues

1. **Name the law.** The central contribution is not just that one factorized
   architecture failed. It is that input factorization does not identify causal
   source. Reafferent attribution needs a gauge-breaking signal.
2. **Make behavior/representation dissociation visible.** A reader should see
   immediately that return and action accuracy pass while absolute self/world
   attribution fails.
3. **Keep the consciousness claim bounded.** The paper is valuable precisely
   because it avoids overclaiming: reafference is a selfhood ingredient, not a
   consciousness certificate.
4. **Tie the result to future agent work.** Long-horizon agents need causal
   boundary tracking for credit assignment, replanning, repair, and probe
   selection. This paper gives the failure mode to avoid.
5. **Clarify the next architecture move.** Active null interventions,
   temporal asymmetry, explicit source labels, or distinct source losses are
   not optional flourishes; they are candidate gauge breakers.

## Rewrite Applied

- Added an abstract-level architecture-law paragraph.
- Added Section 4.4, "Architecture law: reafferent identifiability."
- Added Figure 5, a reafferent-identifiability evidence ladder.
- Reframed the program contribution as a bounded machine-agency law: correct
  action plus separated heads is insufficient for self/world identification.

## Contribution Opportunity

The major contribution is a reusable diagnostic for machine selfhood claims:

> Does the agent have a gauge-breaking signal that pins what it caused, or does
> it merely route behavior through an action-conditioned head?

That question connects reafference, consciousness-adjacent selfhood, and agentic
credit assignment without pretending that the toy system is conscious.
