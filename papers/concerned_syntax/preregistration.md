# Paper 2A Pre-Registration

**Title (working):** Constituency Tests for Concerned Representation in Minimal Agents

**Frozen:** 2026-06-16, before large Modal sweeps.

## Question

Can a minimal concerned agent learn and use causal constituency, rather than a
flat valence map, compression proxy, or generic uncertainty drive?

Arc 1 reached a ceiling where null-only intervention plus shared heads could
not close role-specific mediated identifiability. Arc 2A tests the
interventional side of that ceiling. The central question is not whether an
agent can act correctly on average. The question is whether the agent chooses
an intervention that makes a hidden causal parse knowable, and whether it does
so only when the parse matters for viability.

## Environment

Each trial presents a six-part visible shape. The same visible surface can be
generated by two hidden parse trees. Causal roles only interact when they are
bound inside the same constituent.

High-concern roles:

- `shield` protects `poison` only when both are in the same subtree.
- `repair` repairs `core` only when both are in the same subtree.
- `food` becomes trap-contaminated only when `food` and `trap` are in the same
  subtree.

Low-concern ambiguity:

- `signal` and `ornament` may be structurally ambiguous, but the ambiguity does
  not change viability.

## Selectors

| Selector | Interpretation |
|---|---|
| `null_policy` | never probes |
| `flat_valence` | sees roles but no parse |
| `compression_proxy` | chooses the shortest parse without intervention |
| `uncertainty_only` | probes whenever ambiguity can be reduced |
| `concerned_syntax` | probes when expected parse information is worth its viability cost |

## Interventions

| Intervention | What it reveals |
|---|---|
| `null` | nothing |
| `pair_probe` | whether the causal role pair is bound in the same subtree |
| `distractor_pair_probe` | a matched but non-informative pair relation |
| `high_constituent_move` | which high-level constituent moves as a unit |
| `role_ablation` | the viability signature of the causal constituent |

## Gates

The `concerned_syntax` selector passes the design gate only if all hold:

| Gate | Criterion |
|---|---|
| G1 parse congruity | high-concern parse accuracy >= 0.75 |
| G2 action use | action accuracy >= 0.85 |
| G3 high-concern intervention | high-concern probe rate >= 0.70 |
| G4 no restless inquiry | low-concern probe rate <= 0.25 |
| G5 subtree use | subtree accuracy >= 0.75 |

Negative-control expectations:

- `flat_valence` must fail G1/G5.
- `compression_proxy` must fail the intervention gate.
- `uncertainty_only` must fail G4 by over-probing low-concern ambiguity.

## Interpretation Matrix

| Result | Interpretation |
|---|---|
| Concerned syntax passes; controls fail | task surface separates syntax under concern from flat valence, compression, and uncertainty |
| Uncertainty-only passes | concern threshold too weak; task confounds epistemic curiosity with viable inquiry |
| Compression proxy passes | parse choice is too aligned with shortest-description bias |
| Flat valence passes | causal constituency is not load-bearing enough |
| All fail | intervention language or causal roles are underpowered |

## What This Does Not Claim

This preregistration does not claim consciousness, full agency, or human-like
shape cognition. It defines a minimal benchmark for testing whether causal
constituency is behaviorally and interventionally load-bearing under concern.
