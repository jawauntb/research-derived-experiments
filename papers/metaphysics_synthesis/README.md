## Scope

The synthesis prioritizes:

- structure-compatible model selection under underspecification;
- proxy-resistant finite-agent benchmarks;
- Suite C re-engagement and teacher-free inquiry;
- long-horizon moved-bottleneck and tool-commitment diagnostics;
- concern as metric allocation and vector-valued viability;
- bounded foundation-model and frozen-encoder transport;
- rejected activation-steering claims and other important negative controls.

## Claim Boundary

The paper does not claim consciousness, biological validation, general
autonomy, production reliability, universal OOD certification, open-world
semantic robustness, or broad activation-steering control. It argues for a
finite, testable evaluation rule:

> A system should not pass because it merely gets the answer right; it should
> pass when the future-controlling structure reaches the surface where success
> is chosen.
