# Metric Stack / Concern Geometry Position Package

This folder packages the broad concern-geometry work as a synthesis and
position-paper track, not as the first ICML main submission.

## Role In The Publication Plan

1. Submit or polish the structure-compatible model-selection paper first.
2. Submit or polish the proxy-resistant finite-agent benchmark paper second.
3. Use this package as the broader synthesis:

> Concern-mediated agents improve when viability/error signals are vectorized,
> calibrated against current evidence, coupled to identifying interventions and
> commitment surfaces, and gated by anti-cheat diagnostics that separate
> behavior from internal causal structure.

## Files

- `position_outline.md`: comprehensive structure for a synthesis/arXiv paper.
- `architecture_laws.md`: portable laws extracted from the metric-stack arc.
- `publication_sequence.md`: how this broad paper should follow the narrower
  technical papers.

## Claim Boundary

This package should not claim consciousness, general intelligence, human
behavioral validity, or biological validation. It is about minimal computational
precursors and architecture laws.
