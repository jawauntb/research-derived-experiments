# Position Paper Outline: The Geometry of Concern

## Working Title

The Geometry of Concern: Measurement Laws for Finite Agents

## Thesis

Finite-agent competence becomes more measurable when we ask which structure the
system preserves, which distinctions it allocates capacity to, and whether those
structures still control behavior after world change, intervention, memory
pressure, and OOD shift.

## Abstract Sketch

This position paper synthesizes a multi-study arc on concern-mediated agency in
minimal finite agents. The central claim is not that these agents are conscious
or generally intelligent. The claim is methodological: "what matters" can be
made operational through metric deformation, calibrated inquiry, self/world
attribution gates, commitment-surface memory, and structure-compatible
generalization. Across the arc, behavior-only success repeatedly failed as an
explanation. Each failure forced a new diagnostic or architecture law: preserve
vector-valued concern until decision time; use identifying interventions for
self/world attribution; compute uncertainty against current evidence; cool the
decision to ask without suppressing surprise; bind memory to future action/tool
surfaces; and expose deployment transformations as model-selection pressure.

## Section Plan

1. **Introduction**
   - Meaning/concern as measurable structure, not metaphor.
   - Claim boundary: minimal computational precursors only.

2. **The Metric Stack**
   - Summarize the 20-layer historical stack.
   - Explain why each metric was added after a failure.

3. **The Correction Chain**
   - Behavior is not representation.
   - Representation is not competence.
   - Uncertainty is not error.
   - Current error is not value of probing.
   - Quiet is not stability.
   - Total world prediction is not component identifiability.

4. **Architecture Laws**
   - Convert each failure into a testable design rule.

5. **Connection To Structure-Compatible OOD**
   - Deployment transformations as external structure that must be preserved.

6. **Connection To Proxy-Resistant Agent Benchmarks**
   - Minimum pass rule: behavior + structure gate + anti-cheat controls.

7. **Philosophical Correlates**
   - Keep Heidegger/Gibson/Uexkull/enactive/active-inference material as
     framing, not evidence.

8. **Limitations and Falsification**
   - Simulator-defined viability variables.
   - Privileged null intervention.
   - Small seed counts in older phases.
   - No consciousness or biological validity claim.

9. **Research Program**
   - Teacher-free Suite C.
   - Open-agent/long-horizon transfer.
   - Learned intervention discovery.
   - Larger semantic model-selection suites.
