# Grounded Harness Public D2 Dataset

Sanitized held-out live evaluation rows under the harness-enforced, name-free prompt contract.

## Contents

- `rows.jsonl`: public-schema episode rows only
- `summary.json`: bootstrap contrasts and claim boundary
- `checksums.json`: SHA-256 digests of the published files

## Exclusions

No prompts, transcripts, provider payloads, heuristic CHS candidates, or sealed labels.
