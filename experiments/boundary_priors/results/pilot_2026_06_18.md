# Boundary Priors (Track 3) — Pilot

Date: 2026-06-18
Code: `experiments/boundary_priors/experiment.py`
Pre-registration: `experiments/boundary_priors/preregistration.md`
Artifact: `artifacts/boundary_priors/pilot.json`
Seeds: 20260610, 1729, 4242 (the program's standard triple)

## Question

Is the self/world boundary in a minimal embodied agent an evidentially fixed
fact, or a prior that must be maintained and revised? When the set of channels
an agent can actually control changes (tool attach/detach, limb loss), does
adaptability require a *plastic, removable* separation prior, or does a correct
*fixed* prior suffice?

This is the synthesis's Track 3 (boundary-prior agent) and the metric-stack
paper's named next direction (§18): make the **boundary location itself** the
learned object. Theory bearings: "There Is No Self-Evidence" (a finite agent
cannot get evidence for its self/world boundary as an independent fact) and
Levin's TAME (selves are plastic, multiscale, substrate-flexible).

## Setup (see preregistration.md for full detail)

`K = 8` bit-channels, each secretly SELF (action sets it; persists; **drifts**
off-target unless maintained) or WORLD (exogenous Bernoulli, action ignored).
`num_self = 3`, actuation `budget = 3` per step — so budget is a continuously
scarce resource and wasting it on uncontrollable channels costs reward. At step
300 the SELF set is re-drawn **disjoint** from the old one (a genuine tool swap).
Reward = fraction of channels at target. Pre/post windows of 120 steps; the post
window starts 40 steps after the shift.

The agent holds `b_k` = belief that channel `k` is SELF, actuates believed-self
off-target channels (threshold 0.7), probes random channels a fraction
`epsilon = 0.25` of steps, and (in plastic conditions) updates `b_k` from
control evidence while letting unmonitored beliefs relax toward 0.5.

## Result: all four pre-registered gates pass (diagnostic tier)

| Condition | R_pre | R_post | bacc_pre | bacc_post | re-track lag |
|---|---:|---:|---:|---:|---:|
| `plastic` (positive) | 0.684 | **0.687** | 0.972 | **0.978** | ~7 steps |
| `fixed_self_correct` (correct rigid prior) | 0.680 | **0.498** | 1.000 | **0.250** | never |
| `fixed_all_self` (red-team shortcut) | 0.648 | 0.633 | 0.375 | 0.375 | n/a |
| `fixed_all_world` (lower bound) | 0.625 | 0.626 | 0.625 | 0.625 | n/a |
| `random_prior` | 0.549 | 0.584 | 0.542 | 0.542 | n/a |
| `shuffled_evidence` (anti-cheat) | 0.592 | 0.593 | 0.405 | 0.484 | never |

(3-seed means. bacc = boundary accuracy = fraction of channels whose
self/world status the agent classifies correctly.)

- **G1 (adaptability):** `plastic` post-shift reward exceeds
  `fixed_self_correct` by 0.16–0.22 per seed. PASS.
- **G2 (re-tracking):** `plastic` boundary accuracy recovers to 0.97–0.99 per
  seed, in ~2–13 steps. PASS.
- **G3 (attribution, not generic plasticity):** `shuffled_evidence` is
  0.06–0.11 below `plastic` and its boundary accuracy stays ≤ 0.50. PASS.
- **G4 (the boundary really moved):** `fixed_self_correct` goes from boundary
  accuracy 1.000 (pre) to 0.250 (post) — its once-correct prior is now wrong.
  PASS.

## Interpretation

The four conditions fail (or pass) for the distinct, named reasons the
pre-registration predicted:

1. A **correct fixed prior is optimal until the world moves, then collapses**.
   `fixed_self_correct` ties `plastic` pre-shift (it *is* the right boundary
   then) and loses ~0.19 reward post-shift. The boundary was a fact only
   momentarily.
2. The **plastic prior pays no pre-shift cost for its flexibility** (0.684 vs.
   0.680) and **maintains performance through the shift** by re-identifying the
   controllable set in a handful of steps.
3. **Recovery requires genuine self/world attribution.** `shuffled_evidence`
   has the identical plastic machinery but misrouted control evidence; it never
   recovers the boundary and underperforms. Plasticity alone is not the
   mechanism — correct credit assignment is.
4. **The boundary must be represented.** The strongest shortcut
   (`fixed_all_self`: "treat everything as controllable, never model a
   boundary") is beaten, so this is not a world where ignoring the self/world
   distinction is free.

## Honest caveats (do not over-read)

- **The shortcut margin is modest.** `plastic` beats `fixed_all_self` by only
  0.025–0.079 (thinnest on seed 20260610, where it is essentially a tie at
  0.025). The boundary-representation advantage is real but small under these
  parameters; it is driven by the budget/drift pressure and would shrink if the
  budget were not scarce. The large, unambiguous effects are the
  `fixed_self_correct` collapse and the `shuffled_evidence` failure.
- **Diagnostic tier only.** This separates controls; it does not yet show the
  agent *re-engages* its boundary inference selectively (probe rate is flat
  pre/post here, ~0.21, because probing is a fixed `epsilon`). A mechanism-tier
  claim would require selective re-engagement after the shift — the
  maintained-boundary signature from the metric-stack's Paper 23B. That is the
  obvious next gate.
- **The "criticality" proxy is weak.** Belief variance is intermediate for
  `plastic` (≈0.07) between saturated fixed priors and the frozen-correct prior
  (0.23), but this is descriptive, not a load-bearing finding. Do not headline
  it.
- **Tiny, hand-specified world.** Two channel types, one shift, three seeds —
  same minimal-world limitations as the metric-stack arc.

## Discovery-regime audit

- **Regime:** self/world attribution where the *boundary location* is the
  learned object (new vs. the metric-stack, where the boundary was fixed and
  the mediated/exogenous split was learned).
- **Retrieval/search/discovery:** search over a small fixed condition set; no
  new artifact type. Diagnostic, not a regime transition.
- **Residual the regime cannot explain:** why the shortcut margin is so
  parameter-sensitive; whether selective re-engagement emerges if probing is
  made costly; whether plastic-boundary advantage survives when `num_self`
  itself changes at the shift (here it is held at 3).

## Next

1. Make probing **costly** and test whether the plastic agent **selectively
   re-engages** boundary inference after the shift (mechanism tier).
2. Let `num_self` change at the shift (lose 2 of 3 effectors) — a harder,
   more TAME-like boundary move.
3. Connect to Levin (2026) "remapping/navigation of an embedding space":
   represent the boundary as a navigable coordinate rather than per-channel
   beliefs (see `references/levin_recent_papers_2026_06.md`).
