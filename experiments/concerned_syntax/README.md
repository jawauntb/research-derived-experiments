# Concerned Syntax

Arc 2A asks whether a concerned agent can learn and use causal
constituency, rather than a flat valence map or a compression proxy.

The benchmark is deliberately symbolic and small. A visible six-part shape can
be generated by different hidden parse trees. Causal roles such as shield,
poison, repair, core, food, and trap only interact when they are bound inside
the same constituent. The agent must decide when an intervention is worth its
cost, whether the intervention reveals the parse, and whether the resulting
parse changes action under viability pressure.
