# Store Art

Editable SVG masters and PNG exports for Chrome Web Store review.

| Asset | Size | Purpose |
| --- | ---: | --- |
| `chrome-store-small-tile.png` | 440 x 280 | Required small promotional tile |
| `chrome-store-marquee.png` | 1280 x 800 | Marquee/promotional image |
| `chrome-store-screenshot.png` | 1280 x 800 | Listing screenshot-style product proof |

Regenerate PNGs from the SVG masters with `rsvg-convert` after editing.
